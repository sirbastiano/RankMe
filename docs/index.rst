# RankMe Documentation

Welcome to **RankMe**, a comprehensive PyTorch-based metrics library for machine learning tasks including feature learning, classification, and regression.

## Key Features

- Comprehensive metric suite covering feature learning, classification, and regression
- PyTorch-native implementation with full GPU support  
- Memory-efficient and optimized for large-scale computations
- Easy-to-use API with sensible defaults
